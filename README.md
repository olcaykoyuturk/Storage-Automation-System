# Storage-Automation-System
Bu proje, görüntü işleme (image processing) tekniklerini kullanarak nesneleri tanımlamayı ve envanter yönetimini optimize etmeyi amaçlamaktadır. Görüntü işleme ile tanımlanan nesnelerin sayısı, veritabanı üzerinden takip edilerek stok seviyeleri arttırılmaktadır. Stok yönetimi sürecinde, nesneler AVG (otomatik yönlendirilmiş araç) ile depolama alanlarına taşınmakta ve gerektiğinde depodan istenilen nesne AVG aracılığıyla alınmaktadır. Bu sayede, depo operasyonları daha hızlı, verimli ve hatasız bir şekilde gerçekleşir, envanter yönetimi daha doğru hale gelir.
